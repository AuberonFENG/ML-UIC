## Test Cases
The test folder contains "EfficientNetB0_predict.py" and "VGG16.py" to test functionality.
--'EfficientNetB0_predict.py' is using a default image called "dog.jpg" to classify its class and corresponding probabilities.
--'VGG_16.py' is using VGG module to classify a chosen image and output classes with corresponding probabilities.
